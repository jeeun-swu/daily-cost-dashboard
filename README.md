# 🌟 Daily Cost Board

하루를 **게임처럼 코스트로 관리**할 수 있는 개인 생산성 웹앱입니다.  
하루를 1~5 코스트로 나누고, 외부 일정은 마이너스 코스트로 처리하여  
그날 사용할 수 있는 ‘체력/시간’을 시각적으로 계산할 수 있도록 만든 도구입니다.

개인 루틴 관리, 집중시간 관리, 자기계발용으로 제작했습니다.

---

## 🎮 프로젝트 소개

**Daily Cost Board**는 하루를 RPG 게임처럼 설계할 수 있는 생산성 시스템입니다.

- 1~5 코스트: 난이도·집중도 기반 작업 분류  
- -1~-6 코스트: 외부 일정(알바, 약속, 장보기 등)  
- 코스트 계산을 통한 오늘의 체력 관리  
- 체크한 항목 자동 요약  
- 랜덤 하루 생성(Random Day) 기능  
- 날짜·시간·기온 API 연동

개발·디자인·프로덕트 기획을 모두 직접 수행한 **개인 미니 프로젝트**입니다.

## ✨ 주요 기능

### ✔ 1. 코스트 기반 작업 시스템
- 1코: 가벼운 행동(물 마시기, 정리 등)
- 2코: 20~30분 작업(단어, 노션 정리 등)
- 3코: 집중 1시간
- 4코: 심화 작업
- 5코: 특대형 작업(2~3시간)

### ✔ 2. 외부 일정 = 마이너스 코스트
- 약속, 데이트, 알바, 이동시간 등  
- -1 ~ -6 코스트  
- 외부 일정이 많은 날은 자동으로 이용 가능 코스트 감소

### ✔ 3. 체크 리스트 + 자동 요약 패널
- 체크한 항목만 오른쪽 패널에 정리  
- 항목별 삭제(❌) 가능  
- 코스트 순 정렬 (오름/내림차순)

### ✔ 4. 랜덤 하루 생성 (Random Day)
- 남은 코스트 내에서 가능한 행동 조합  
- 즉시 ‘오늘의 미션’ 생성  
- 모든 체크/요약 자동 반영

### ✔ 5. 날짜·시간·기온 자동 업데이트
- 날짜: 📅 YYYY년 MM월 DD일 (요일)
- 시간: ⏰ HH:MM (실시간 갱신)
- 기온: 🌤 °C (Open-Meteo API)

## 🧩 개발 과정 & 문제 해결 기록

### 1) 체크 UI 색상 변화 문제  
- 체크했는데 스타일이 적용되지 않는 버그  
- ✔ `syncLabelClasses()` 함수 제작 → 색상 자동 적용

### 2) 코스트 초과 시 체크가 막히는 문제  
- 처음에는 초과하면 클릭 안 됨  
- ✔ 사용자 경험을 위해 “체크는 되지만 빨간색 경고 표시”로 개선

### 3) 요약 패널에서 ❌ 클릭 시 체크박스와 불일치
- 삭제해도 원래 항목이 그대로 체크되어 있는 문제가 있었음  
- ✔ ❌ 클릭 → 체크 해제 → 스타일 제거 → 요약 재렌더 순으로 해결

### 4) 랜덤 기능 동작 안 함  
- UI는 업데이트 안 되고 내부 값만 변하는 버그  
- ✔ 랜덤 이후  
  - updateStatus()  
  - updateSummary()  
  - syncLabelClasses()  
  호출하도록 수정

### 5) 날짜 정보 박스가 레이아웃에서 밀리는 문제  
- flex/grid 내부에 포함되어 UI가 깨짐  
- ✔ 요약 패널 옆 ‘정보 박스’를 별도 독립 카드로 생성해서 해결

### 6) 기온 API 오류  
- 위치 권한 거부 시 error 발생  
- ✔ fallback 메시지 추가 + 날짜/시간은 항상 출력되도록 처리
